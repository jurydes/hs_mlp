# Perceptron

Реализация перцептрона Розенблатта на Haskell — алгоритм бинарной классификации.

## Функционал

- Обучение на данных из текстового файла
- Три функции активации: `step`, `sign`, `sigmoid`
- Предсказание меток для тестовой выборки
- Вывод точности (accuracy)
- Сохранение и загрузка обученной модели

## Формат данных

Каждая строка файла — один объект: числа через пробел, последнее — метка класса (0 или 1).  
Строки, начинающиеся с `#`, игнорируются.

```
# пример
5.1 3.5 1.4 0.2 0
7.0 3.2 4.7 1.4 1
```

## Запуск

```bash
cabal run perceptron -- <обуч.файл> <тест.файл> [файл_модели] [step|sign|sigmoid]
```

**Примеры:**

```bash
# обучить и оценить
stack run perceptron -- data/train.txt data/test.txt

# с сохранением модели и выбором функции активации
stack run perceptron -- data/train.txt data/test.txt model.txt sigmoid
```

Если `файл_модели` указан и существует — модель загружается из него. Иначе обучается с нуля и сохраняется.
